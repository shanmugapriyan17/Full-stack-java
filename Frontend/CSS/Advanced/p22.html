CSS PSEUDO-CLASSES..!

The CSS pseudo-classes allow you to style the dynamic states of an element such as hover, active and focus state, as well as elements that are existing in the document tree but can't be targeted via the use of other selectors without adding any IDs or classes to them, for example, targeting the first or last child elements or we can target middle elements also

A pseudo-class starts with a colon (:). Its syntax can be given with:

selector:pseudo-class { property: value; }
