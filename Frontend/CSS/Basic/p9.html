Defining color value

Color is CSS most often specifies in the folllowing formats:

  1. A color keyword - like - " red ", "Green", "blue", "transparent ", etc..
  2. A hex and hexa value -like '#ff0000', "#00ff00" hexa - #ff00050" (50 is alpha to make transparent we need to use a value in hexa between 01-99) ,etc
  3. An RGB alpha to make transparent in (RED, GREEN, BLUE) = like "rgb(255,0,0)" and RGBA - a is alpha to make transparent in rgb we need to use a value below 1
